## ✨ What's Cloned

- 🔵 **Navbar** — Flipkart logo, search bar, login button, and cart icon
- 📂 **Category bar** — Horizontal scrollable category navigation links
- 🖼️ **Hero banner** — Promotional deal banners with image layout
- 🛍️ **Product grid** — Product cards with images, names, and prices
- 🖼️ **Assets** — Custom images stored in the `img/` folder
- 📄 **Footer** — Multi-column footer with links and branding

## 🛠️ Getting Started

### Prerequisites

Just a modern web browser — no installs needed!

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/ankushkumar01/Flipkart-Homepage-Clone.git

# 2. Navigate into the folder
cd Flipkart-Homepage-Clone

# 3. Open in your browser
open index.html
# or double-click index.html in your file explorer
```
